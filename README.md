# 脑转移肿瘤BED10与1年/2年肿瘤控制关系研究

## 项目概述
本项目基于脑转移肿瘤患者的临床数据，通过逻辑回归分析BED10与1年、2年肿瘤控制的关联，并利用XGBoost模型识别影响长期控制的关键指标，旨在为脑转移肿瘤放疗方案优化提供数据支持。

## 数据集说明
### 1. 数据来源
脑转移肿瘤患者的临床诊疗记录及随访数据（包含基线信息、治疗参数、疗效评价结果）。
